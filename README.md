# 📐 Math-api-app

> Aplicativo mobile de estudo de matemática desenvolvido com React Native e TypeScript, que consome uma API externa para gerar operações matemáticas e permitir que usuários pratiquem cálculos.

## Funcionalidades

- Geração de contas matemáticas via API
- Prática de operações matemáticas
- Interface mobile intuitiva

## Tecnologias

- React Native
- TypeScript
- API de geração de operações matemáticas
